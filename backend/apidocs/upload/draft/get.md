# 获取编辑中的草稿文档
获取编辑中的草稿文档

**URL** : `/api/upload/draft`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : 仅管理员

## Success Response

**Code** : `200 OK`

**Content examples**



```json
  {
    "id": "草稿内容",
    "content": "草稿内容",
    "createdAt": "2024-08-12T07:37:03.771Z",
    "updatedAt": "2024-08-12T07:37:03.771Z",
  },
```
